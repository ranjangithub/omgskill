## Structure of each registry

```
## RSS Feeds — AI & Technology Core     # shared cross-industry feeds
## RSS Feeds — {Industry} Specific      # industry trade feeds
## RSS Feeds — Research & Policy        # regulatory + academic feeds
## Key Publications (Manual)            # paywalled / no RSS but worth monitoring
## Newsletters                          # subscribe-worthy email signals
## Regulatory & Standards Bodies        # where rules come from
## High-Signal LinkedIn Accounts        # key people to follow
## Notes                                # feed quality warnings + caveats
```

## Updating registries

Edit the relevant `source-registry.md` directly, or use the omgskill admin page
at `/dashboard/admin` to add/edit/delete entries without touching markdown manually.
